Linear Equations & Systems

Topic id gmat::quant::algebra::linear_equations · Domain Algebra · Prereqs algebraic expressions · ~22 min

Learning objectives

  • Solve one-variable linear equations, including those with parentheses and fractions.
  • Use inverse operations in the right order.
  • Solve two-variable systems by substitution and by elimination.
  • Translate simple statements into linear equations.

Model: I do → we do → you do · application-first (explanations revealed after an attempt) · Rosenshine (daily review), Archer (I-do/we-do/you-do), SPOV2 application-first.

1Opening — retrieval & bridge

Builds on: Combining like terms and distributing.

Do Now · from memory, no notes

From memory, no calculator. 4 minutes.

  1. Solve 2x + 5 = 13.x = 4.
  2. Solve x/3 = 6.x = 18.
  3. Solve 3(x − 2) = 9.x = 5.
  4. If x + y = 10 and x = 4, find y.y = 6.
  5. In 2x + 5 = 13, why subtract 5 before dividing by 2?Undo operations in reverse (inverse) order — addition last applied is undone first.

Retrieval starter (5 min). Retrieves the isolate-the-variable routine and the substitution idea that drives systems today.
If students struggle: If order trips them, label the operations done TO x, then undo them last-in-first-out.

Prior-knowledge bridge

Solving an equation is just undoing operations in reverse, as Q5 highlights. A system is two equations sharing the same unknowns; the trick is to remove one variable — by substitution (use one equation inside the other) or elimination (add/subtract to cancel) — and reduce it to the single-variable problem you already own.

Learning intention

By the end you can solve linear equations (including fractional ones) and two-variable systems by substitution or elimination.

You'll know you've got it when:

  • I can isolate a variable using inverse operations in the correct order.
  • I can clear fractions before solving.
  • I can solve a 2×2 system and check the solution in both equations.

2I do — worked example

Watch the reasoning modeled end to end before you try.

I DO · worked examplemedium
Solve for x: 5x − 3 = 2x + 12.
  • A3
  • B5
  • C9
  • D15
  • E−5

Think-aloud

  1. Move variable terms to one side: subtract 2x from both → 3x − 3 = 12.
  2. Move constants to the other side: add 3 → 3x = 15.
  3. Divide by the coefficient: x = 15/3 = 5.
  4. Check: 5(5) − 3 = 22 and 2(5) + 12 = 22. ✓

Correct: B. Collect variables: 5x − 2x = 12 + 3 → 3x = 15 → x = 5.

Takeaway. Gather variable terms on one side and constants on the other, then divide by the coefficient.
