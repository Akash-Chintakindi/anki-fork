Inequalities

Topic id gmat::quant::algebra::inequalities · Domain Algebra · Prereqs linear equations · ~22 min

Learning objectives

  • Solve linear inequalities using inverse operations.
  • Reverse the inequality sign when multiplying or dividing by a negative.
  • Solve and interpret compound inequalities.
  • Count integer solutions within a range.

Model: I do → we do → you do · application-first (explanations revealed after an attempt) · Rosenshine (daily review), Archer (I-do/we-do/you-do), SPOV2 application-first.

1Opening — retrieval & bridge

Builds on: Solving linear equations by inverse operations.

Do Now · from memory, no notes

From memory, no calculator. 4 minutes.

  1. Solve x + 4 < 9.x < 5.
  2. Is −3 < −1 true?Yes (−3 is farther left).
  3. Solve 2x > 10.x > 5.
  4. If x > 3, is x = 3 a solution?No — strict inequality excludes 3.
  5. Solve −x < 4. Why does it become x > −4?Multiplying/dividing by a negative reverses the inequality.

Retrieval starter (5 min). Retrieves equation-style solving and surfaces the single rule that makes inequalities different: flip when you multiply or divide by a negative.
If students struggle: If the flip is unintuitive, test x = −10 in −x < 4: −(−10) = 10, which is NOT < 4, confirming the reversed solution.

Prior-knowledge bridge

Inequalities solve exactly like equations — with one twist from Q5: multiplying or dividing both sides by a negative reverses the direction. Master that and you also handle compound inequalities (a range) and 'how many integers' questions the GMAT likes.

Learning intention

By the end you can solve linear and compound inequalities, flipping the sign correctly, and count integer solutions in a range.

You'll know you've got it when:

  • I solve inequalities with inverse operations like equations.
  • I reverse the sign exactly when I divide/multiply by a negative.
  • I can read a compound inequality as a range and count its integers.

2I do — worked example

Watch the reasoning modeled end to end before you try.

I DO · worked examplemedium
Solve for x: −3x + 2 ≤ 11.
  • Ax ≤ −3
  • Bx ≥ −3
  • Cx ≤ 3
  • Dx ≥ 3
  • Ex ≤ 9

Think-aloud

  1. Subtract 2 from both sides: −3x ≤ 9.
  2. Divide by −3 — because it's negative, reverse ≤ to ≥: x ≥ −3.
  3. Check x = 0: −3(0) + 2 = 2 ≤ 11 ✓ and 0 ≥ −3 ✓.

Correct: B. Subtract 2: −3x ≤ 9. Divide by −3 and FLIP the sign: x ≥ −3.

Takeaway. Solve like an equation, but reverse the inequality whenever you multiply or divide by a negative number.

3We do — guided practice

Scaffolds and immediate feedback are one click away. Fade them as you gain confidence.

we_1 · we do (guided)easy
Solve: 4x − 7 > 5.
  • Ax > 3
  • Bx < 3
  • Cx > 12
  • Dx > −3
  • Ex < 12
Scaffold hints (peek only if stuck)
  1. Add 7 to both sides.
  2. 4x > 12.
  3. Divide by positive 4 — sign stays.
Immediate feedback & explanation

If correct: Right — x > 3 (no flip, since 4 is positive).
If incorrect: You only flip for a negative divisor; here it's +4.

Answer: A. Add 7: 4x > 12. Divide by +4 (no flip): x > 3.

we_2 · we do (guided)medium
Solve: 5 − 2x ≥ 1.
  • Ax ≤ 2
  • Bx ≥ 2
  • Cx ≤ −2
  • Dx ≥ −2
  • Ex ≤ 3
Scaffold hints (peek only if stuck)
  1. Subtract 5: −2x ≥ −4.
  2. Dividing by −2 reverses ≥ to ≤.
  3. x ≤ 2.
Immediate feedback & explanation

If correct: Yes — x ≤ 2.
If incorrect: The −2 divisor forces the flip; ≥ becomes ≤.

Answer: A. Subtract 5: −2x ≥ −4. Divide by −2 and flip: x ≤ 2.
