Quadratic Equations

Topic id gmat::quant::algebra::quadratics · Domain Algebra · Prereqs algebraic expressions (factoring), linear equations · ~24 min

Learning objectives

  • Factor a quadratic of the form x^2 + bx + c.
  • Solve quadratics using the zero-product property.
  • Solve x^2 = kx without losing the x = 0 root.
  • Use the relationships sum of roots = −b and product of roots = c (for monic quadratics).

Model: I do → we do → you do · application-first (explanations revealed after an attempt) · Rosenshine (daily review), Archer (I-do/we-do/you-do), SPOV2 application-first.

1Opening — retrieval & bridge

Builds on: Factoring expressions and solving linear equations.

Do Now · from memory, no notes

From memory, no calculator. 4 minutes.

  1. Factor x^2 + 5x + 6.(x + 2)(x + 3).
  2. Expand (x + 4)(x − 1).x^2 + 3x − 4.
  3. Solve x(x − 3) = 0.x = 0 or x = 3.
  4. Factor x^2 − 16.(x − 4)(x + 4).
  5. If (x − 2)(x + 5) = 0, why must a factor be 0?A product equals 0 only if at least one factor is 0.

Retrieval starter (5 min). Retrieves trinomial factoring, difference of squares, and the zero-product property — the three tools that solve almost every GMAT quadratic.
If students struggle: If factoring stalls, list factor pairs of c and test which pair sums to b.

Prior-knowledge bridge

You can already factor (last lesson) and solve linear equations. Q5 is the hinge that connects them: once a quadratic is factored and set equal to 0, the zero-product property turns it into two linear equations. Today you'll factor, then solve — and learn the trap of dividing by x.

Learning intention

By the end you can solve quadratic equations by factoring and the zero-product property, and read off the sum and product of the roots.

You'll know you've got it when:

  • I can factor x^2 + bx + c by finding two numbers with product c and sum b.
  • I can apply the zero-product property to get both solutions.
  • I move everything to one side instead of dividing by x.

2I do — worked example

Watch the reasoning modeled end to end before you try.

I DO · worked examplemedium
Solve for x: x^2 − 7x + 10 = 0.
  • Ax = 2 or x = 5
  • Bx = −2 or x = −5
  • Cx = 1 or x = 10
  • Dx = 2 or x = −5
  • Ex = −2 or x = 5

Think-aloud

  1. Need two numbers multiplying to +10 and adding to −7 → −2 and −5.
  2. Factor: (x − 2)(x − 5) = 0.
  3. Zero-product property: x − 2 = 0 or x − 5 = 0.
  4. So x = 2 or x = 5.

Correct: A. Two numbers with product +10 and sum −7 are −2 and −5: (x − 2)(x − 5) = 0, so x = 2 or x = 5.

Takeaway. For x^2 + bx + c = 0, find two numbers multiplying to c and adding to b; then set each factor to 0.

3We do — guided practice

Scaffolds and immediate feedback are one click away. Fade them as you gain confidence.

we_1 · we do (guided)medium
Solve: x^2 + 2x − 15 = 0.
  • Ax = 3 or 5
  • Bx = −5 or 3
  • Cx = 5 or −3
  • Dx = −5 or −3
  • Ex = 15 or −1
Scaffold hints (peek only if stuck)
  1. Find two numbers with product −15 and sum +2.
  2. +5 and −3 work.
  3. (x + 5)(x − 3) = 0.
Immediate feedback & explanation

If correct: Right — x = −5 or 3.
If incorrect: A negative product means opposite signs; the larger absolute value (5) carries the sign of b (+).

Answer: B. Product −15, sum +2 → +5 and −3: (x + 5)(x − 3) = 0, so x = −5 or x = 3.

we_2 · we do (guided)medium
Solve: x^2 = 9x.
  • Ax = 9
  • Bx = 0 or 9
  • Cx = 3 or −3
  • Dx = 0 or −9
  • Ex = 81
Scaffold hints (peek only if stuck)
  1. Don't divide by x — that throws away a root.
  2. Bring all terms to one side: x^2 − 9x = 0.
  3. Factor out x: x(x − 9) = 0.
Immediate feedback & explanation

If correct: Yes — both x = 0 and x = 9.
If incorrect: Dividing by x loses x = 0. Factor instead.

Answer: B. Move everything to one side: x^2 − 9x = 0 → x(x − 9) = 0 → x = 0 or x = 9.
